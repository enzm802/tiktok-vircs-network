## TikTok 的风控到底在看什么

很多人以为挂个代理就算"美国IP"了。平台不是这么看的。

TikTok 风控系统会检查以下几个维度：

**IP 类型**：住宅宽带、商业数据中心、机场节点——对平台来说是三类完全不同的信号。普通 VPS 机房的 IP，ASN 归属是数据中心，一眼就能识别出来。

**IP 纯净度**：一个 IP 被多少人用过？有没有投诉记录？有没有出现在黑名单里？共享代理的 IP 纯净度通常很低，用的人越多越容易被标记。

**登录一致性**：注册时 IP 在洛杉矶，上传产品时跳到纽约，直播时又换了新加坡——这种"物理瞬移"会直接触发异常判定。平台希望你的网络行为看起来像一个真实的本地用户，而不是到处乱跳的工具账号。

**设备指纹一致性**：IP 只是其中一层，浏览器指纹、设备 ID、操作习惯也在采集范围内。但 IP 是基础，其他层都建立在这之上。

## 住宅IP、数据中心IP、机场节点——到底有什么区别

这三个概念经常被混用，但对 TikTok Shop 运营来说，差异非常大。

**数据中心IP（机房IP）**：价格便宜，速度快，但 WHOIS 查询归属是数据中心 ASN，平台一眼就能识别不是真实用户。适合跑脚本、测试接口，不适合社媒运营主账号。

**机场节点/共享代理**：本质是很多人共用一条出口线路，IP 轮换频率高，纯净度往往很差。用这类环境登录 TikTok 主账号，被限流是早晚的事。

**住宅IP**：来自真实的家庭宽带网络，ISP 归属是 Comcast、AT&T 这类居民互联网服务商。对平台来说，这个 IP 看起来就是一个普通美国居民在家上网。

在住宅 IP 里，还有一个更细的分类：

- **静态住宅IP**：固定不变，每次登录都是同一个 IP，账号历史轨迹连贯。适合主账号长期运营、店铺后台、广告账户。
- **动态住宅IP**：自动轮换，适合辅助任务，不建议用于主账号持续登录。

对 TikTok Shop 来说，静态住宅 IP 是最接近"理想TikTok Shop网络环境"的选择。
